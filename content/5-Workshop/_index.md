# TSL-SignMap - Traffic Sign Management System

#### Overview

**TSL-SignMap** is a community-driven traffic sign location management system built on a **Serverless architecture on AWS**.  
The system serves three user groups: **Admin, Contributors, and Drivers**, featuring crowdsourcing, AI detection, and voting system.

In this workshop, you will learn how to design, configure, and deploy a complete crowdsourcing mobile backend on AWS using modern cloud services.
